## 使用規則

1. **記錄時機**：使用者拍板 / 主場景拍板新情境（§1.4 處理流程走完）→ 立即追加一列
2. **欄位**：
   - **ID**：`CA-001` / `CA-002` ... 流水編號
   - **日期**：`YYYY-MM-DD`
   - **情境**：1 行問題描述
   - **決策**：1 行選擇結果
   - **來源**：`t-xxxxxxxx seq=N` 或討論出處
   - **備註 / 狀態**：選擇題關鍵背景 / 升級後標「已升級 → 見 X.Y」/ 廢止標「廢止，理由」
3. **整理時**：
   - 升級 → 寫進規範正文（collab-framework 1/2/3 或 DECISIONS / coding-guidelines 等）+ 本檔該列備註「已升級 → 見 X.Y」
   - 廢止 → 本檔該列備註「廢止，理由」
   - 整理完可保留歷史（供後續追溯）或使用者明示清理
